Analyze the current recruiting pipeline for bottlenecks and opportunities.

1. Check config exists. If not, run /setup.
2. Pull pipeline data from ATS (or mock data).
3. Calculate and display:

**Pipeline by Stage**
Sourced > Screened > Submitted > Interviewing > Offered > Placed
Show count at each stage and conversion rate between stages.

**Bottleneck Detection**
Where are candidates getting stuck? Flag stages with below-average conversion or above-average time.

**Velocity Trends**
Time spent in each stage. Compare to last week/month. Improving or declining?

**Fill Rate Analysis**
By job type, by client, by recruiter. Who/what is performing above and below average?

**Recruiter Utilization**
Reqs per recruiter. Who is overloaded? Who has capacity?

4. Highlight top 3 actionable items (e.g., "15 candidates stuck in Submitted for 10+ days — follow up with these clients").
5. Ask: "Want to drill into any specific metric, client, or recruiter?"
